**Logic Gates Simulation using Verilog (Vivado)**

**📌 Project Overview**

 This project demonstrates the implementation and simulation of basic logic gates using Verilog HDL in Xilinx Vivado.

 The design includes fundamental gates like AND, OR, and NOT, and verifies their functionality using a testbench and waveform simulation.

**🛠️ Tools & Technologies**
 💻 Xilinx Vivado 2024.1

 🔤 Verilog HDL

 📊 Simulation Waveform Viewer

**⚙️ Design Description**
**Inputs:**

a : Input 1

b : Input 2
**Outputs:**

y0 : AND Gate Output

y1 : OR Gate Output

y2 : NOT Gate Output

**▶️ Simulation**

The testbench applies different combinations of inputs:

**Time     a   b    AND (y0)  OR (y1)  NOT (y2)**

0 ns     0   0     0          0        1
10 ns    0   1     0          1        1
20 ns    1   0     0          1        0
30 ns    1   1     1          1        0
